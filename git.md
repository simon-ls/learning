### New Branch
1. git chekout master
2. git pull --all
3. git checkout -b feature/....
4. git commit ....
5. git push (https://help.github.com/articles/pushing-to-a-remote/)

### Merge branch in
1. git checkout [parent]
2. git pull --all
3. git merge [child]
4. gulp and resolve
5. git push
6. repeat for epic and dev/staging
